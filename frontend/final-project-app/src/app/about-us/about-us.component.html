About Group 12's Final Project

Our Site

The purpose of our site is to display real-time data from Augur. Specifically, the different repositories and groups that are contained in Augur and the amount of pushes/pull, additions, issues, groups and their contributions.

Software

Front-End

In order to make our front-end UI and user friendly as possible, we decided to implement the application Angular Material UI component. In doing this, it allowed us to not only easily apply the different components that Material offered, but also include a strong since of theme and unison across each of the pages and tabular navigation bar. Ionic also makes it easy for the user to use our website on their phones or any other mobile device.


Back-End

Originally being a Full Stack Group, we had originally planned to implement Firebase as both our server and also our database. Using our HTTPClient calls to the Augur API, we would be able to store the data retrieved in our Firebase database, and then make calls to our database in order to display the data on our UI.

Transitioning to Front End

When becoming a Front-End Group, we decided to not implement the Firebase database as originally planned since it was unnecessary, and instead focus on displaying real-time data clearly on multiple pages and the overall Ionic application.
